Welcome to GNNHap Data Visualization Server

GNNHap is an AI-driven genetic discovery program for Inbred mice.

Usage

  1. Input a new gene name (Human Gene Symbol) in the TextInputBox to update everything in this page
  2. Input a new MeSH term in the TextInputBox to update everything in this page
  3. Save DataTable by click download button.
  4. Save Figures by click save widget below the plots.
