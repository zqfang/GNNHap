Welcome to GNNHap Data Visualization Server

GNNHap is an AI-driven genetic discovery program for Inbred mice.

Usage

  1. Input a new dataset name in the TextInputBox to update everything in this page
  2. Select MeSH term to get literature score
  3. Click DataTable row to highlight a gene
  4. Click DataTable header to sort a column. Clicking one more time changes sort direction.
    Use Shift + click to sort multiple columns simultaneously.
    Use Ctrl + click to return to natural order.
  5. Save DataTable by click download button.
  6. Save Figures by click save widget below the plots.
